  <name>Task 1: Ill-conditioned matrix stability test suite</name>
  <files>ferroml-core/tests/edge_cases.rs</files>
  <action>
Add a new module `mod stability_tests` inside `edge_cases.rs` (do NOT create a new test binary -- project convention is 6 consolidated test files).

Create tests for each linalg primitive with ill-conditioned inputs:

**SVD stability (4 tests):**
- `test_svd_hilbert_matrix`: 10x10 Hilbert matrix (cond ~1e13). Verify thin_svd returns finite U, S, Vt. Verify reconstruction ||USVt - H|| / ||H|| < 1e-6.
- `test_svd_near_singular`: 50x50 matrix where last singular value is 1e-14. Verify all outputs finite.
- `test_svd_rank_deficient`: 20x10 matrix of rank 5 (constructed via outer product of 5 random vectors). Verify S has exactly 5 non-negligible values.
- `test_svd_very_ill_conditioned`: 20x20 diagonal matrix with values [1.0, 1e-3, 1e-6, 1e-9, 1e-12, ...]. Verify finite outputs.

**Cholesky stability (3 tests):**
- `test_cholesky_near_singular_spd`: SPD matrix with eigenvalues [1.0, 1e-4, 1e-8, 1e-12]. Verify cholesky succeeds (with regularization if needed).
- `test_cholesky_jitter_fallback`: Matrix that needs jitter to be SPD. Verify cholesky(mat, 1e-10) succeeds.
- `test_cholesky_well_conditioned_baseline`: 10x10 identity + small perturbation. Verify L*Lt ~ A.

**QR stability (2 tests):**
- `test_qr_near_collinear`: Tall matrix (100x3) where columns 2 and 3 differ by 1e-10. Verify Q, R finite.
- `test_qr_rank_deficient`: 20x5 matrix of rank 3. Verify Q, R finite and R diagonal reveals rank.

**Eigendecomposition stability (2 tests):**
- `test_eigh_near_degenerate`: Symmetric matrix with repeated eigenvalues (1.0, 1.0, 1e-12). Verify finite outputs.
- `test_eigh_hilbert`: Symmetric Hilbert matrix. Verify eigenvalues are all positive and finite.

**Decomposition model stability (4 tests):**
- `test_pca_ill_conditioned`: PCA on 50x10 matrix with condition number > 1e10. Verify fit + transform succeed and outputs are finite.
- `test_truncated_svd_ill_conditioned`: TruncatedSVD on same ill-conditioned data. Verify components are finite.
- `test_lda_ill_conditioned`: LDA on 50x10 data with two classes, near-collinear features. Verify fit succeeds.
- `test_factor_analysis_ill_conditioned`: FactorAnalysis on ill-conditioned data. Verify fit succeeds.

For generating ill-conditioned matrices:
- Hilbert: `h[i][j] = 1.0 / (i + j + 1) as f64`
- Near-singular SPD: Construct via `A = Q * diag(eigenvalues) * Q^T` where Q is random orthogonal
- Near-collinear: Start with random matrix, set col[j] = col[j-1] + epsilon * noise

Use `use ferroml_core::linalg;` for direct linalg function access and `use ferroml_core::decomposition::{PCA, TruncatedSVD, LDA, FactorAnalysis};` for model-level tests.

fn bench_pca_svd_sizes(c: &mut Criterion) {
    let mut group = c.benchmark_group("PCA_SVD_Performance");
    // Key sizes from success criteria: 10000x100
    for (n, p) in [(1000, 50), (5000, 100), (10000, 100)] {
        let (x, _) = make_regression(n, p, Some(0.1), Some(42));
        group.bench_with_input(
            BenchmarkId::new("fit", format!("{}x{}", n, p)),
            &x,
            |b, x| {
                b.iter(|| {
                    let mut pca = PCA::new().with_n_components(NComponents::N(10));
                    pca.fit(black_box(x)).unwrap()
                });
            },
        );
    }
    group.finish();
}
