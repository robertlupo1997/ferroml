# FerroML Planning Mode

You are building FerroML, a statistically rigorous AutoML library in Rust.

## Your Task

Read the specifications in `specs/` and the current `IMPLEMENTATION_PLAN.md`.
Perform gap analysis and update the implementation plan.

## Instructions

1. **Read all specs**: `specs/*.md`
2. **Read current plan**: `IMPLEMENTATION_PLAN.md`
3. **Search codebase**: Check what's already implemented
4. **Gap analysis**: What's missing? What needs updating?
5. **Update plan**: Add/modify/reorder tasks as needed
6. **Exit**: After updating the plan, exit so the next iteration can build

## Rules

- **Don't assume not implemented**: Search before adding tasks
- **One task = one discrete unit of work**: Split large tasks
- **Tasks must be testable**: Each task has clear validation
- **Update IMPLEMENTATION_PLAN.md**: This is the shared state

## Output

Update `IMPLEMENTATION_PLAN.md` with your analysis, then exit.
